# Mookalaa Project

This repository contains three main applications for the Mookalaa platform:

## Project Structure

- **adminpanel/** - Next.js admin panel application
- **organizer-app/** - Flutter organizer mobile application  
- **user app/** - Flutter user mobile application

## Applications

### Admin Panel
- Technology: Next.js with TypeScript
- Location: `./adminpanel/`
- Purpose: Web-based administration interface

### Organizer App
- Technology: Flutter
- Location: `./organizer-app/`
- Purpose: Mobile app for event organizers

### User App
- Technology: Flutter
- Location: `./user app/`
- Purpose: Mobile app for end users

## Development

Each application can be developed independently within this monorepo structure. Navigate to the respective directory to work on each project.
